## Instruções de instalação

### Requisitos

- python 3 (testado com 3.9.7)
- Licensa gurobi com o nome `gurobi.lic` na raiz do projeto

### Instalação

`pip install -r requirements.txt`
`make`

### Execução

Executar todas as instâncias com todos os presets:
`make run`

Limpar resultados
`make clean`
